# Projede Kullanılan Teknolojiler

## Programlama Dilleri
- Python 3
- Bash (Shell Script)
- HTML5
- CSS3
- JavaScript (Vanilla)
- Arduino (kapi_kontrol.ino)

## Kütüphaneler ve Modüller
- Python Standart Kütüphaneleri (os, sys, sqlite3, subprocess, vb.)
- PyQt5 (GUI için, bazı pencerelerde)
- SMB protokolü için pysmb veya subprocess ile smbclient

## Sistem ve Paketleme
- systemd (Linux servis yönetimi)
- polkit (Yetkilendirme kuralları)
- Debian Paketleme (DEBIAN/control, postinst, prerm, postrm)
- .desktop dosyası (Uygulama kısayolu)
- dpkg (Debian paket kurulumu)

## Web
- Offline HTML/CSS/JS arayüz (cevrimdisi.html)
- Google Fonts yerine yerel font dosyaları (Montserrat, Exo2, vb.)

## Diğer
- SQLite (veritabani.py)
- Font dosyaları (TTF)
- systemctl, journalctl (servis yönetimi ve log)

## Donanım
- Arduino tabanlı kapı kontrolü (otomasyon/kapi_kontrol.ino)

---
Bu liste, projenin kaynak kodu ve paketleme yapısı incelenerek hazırlanmıştır. Ekstra bağımlılıklar veya özel teknolojiler için README ve script dosyalarına bakınız.
